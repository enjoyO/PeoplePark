## 功能介绍 
    
中山公园预约小程序，由于入园市民、游客较多，为更好保障疫情防控和游客游园安全，公园将采取网上预约方式控制入园游客在可容峰值内。准备公园游玩的市民游客，按小程序首页提示进行预约，预约成功即可入园。进入公园时，须戴好口罩，主动扫描安康码，并向工作人员出示安康码及预约登记信息，在工作人员指引下，经入园导向通道，排队入园
